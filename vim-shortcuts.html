Vim Shortcuts for Happier Python Coding

Posted by Bob on Mon 23 January 2017 in Tools

In this article I show you 5 .vimrc shortcuts to speed up your Python development.

Save and run Python

nmap ,p :w:!python3 %

For p

nmap ,2 :w<CR>:!python2.7 %<CR>

Flake 8 check

As featured in our pep8 article:

autocmd FileType python map <buffer> ,f :call Flake8()<CR>

ConqueTermVSplit

I played with this long time ago and it requires this plugin.

To open a Python in vertical split: 

nmap cp :ConqueTermVSplit python3<CR>

Similarly you can use q: to open up a command-line window or (slower) cntrl-z to put your code in the background.

Navigate files

I use NERD tree which opens a nice file tree you can navigate with regular Vim strokes, I mapped it to cntrl-x:

map <C-x> :NERDTreeToggle<CR>

Spacing

Highlighted here and from RealPython's great guide VIM and Python - a Match Made in Heaven:

au BufNewFile,BufRead *.py
    \ set tabstop=4
    \ set softtabstop=4
    \ set shiftwidth=4
    \ set textwidth=79
    \ set expandtab
    \ set autoindent
    \ set fileformat=unix

Bonus .bashrc alias

As soon as I start a new project I run pvenv to create and start a new virtual env:

alias pvenv='python -m venv venv && source venv/bin/activate'

Keep Calm and Code in Python!
